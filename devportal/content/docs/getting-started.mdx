## Prerequisites

- Docker
- A Flow access node endpoint (gRPC)

## Run Locally (Docker Compose)

From the repository root:

```bash
docker compose up --build
```

Services:
- Postgres: `localhost:5432`
- Backend API: `localhost:8080`
- Frontend UI: `http://localhost:5173`
- Developer Portal: `http://localhost:3000`

## Validate

- Health: `GET /api/health` (via the main UI domain) or `GET /health` (direct backend)
- OpenAPI:
  - `GET /api/openapi.yaml`
  - `GET /api/openapi.json`

## Common Issues

- History backfill needs historic/archive nodes to index blocks prior to the latest spork root.
- CoinGecko may return `429` when rate limited; the UI should degrade gracefully.
