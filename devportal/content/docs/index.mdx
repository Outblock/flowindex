## OpenAPI Spec

You can fetch the OpenAPI document (proxied through this portal) at:

- YAML: `/flowscan-api/openapi.yaml`
- JSON: `/flowscan-api/openapi.json`

## Notes

- The API reference uses a built-in proxy (`/flowscan-api/*`) to avoid CORS issues.
- Authentication / API keys are not implemented yet. Default public rate limit applies.
